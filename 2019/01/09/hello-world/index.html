Hexo

hexo来搭建基于github服务器的博客是真的快速。

只需要有git环境就成，其他的所有HTML和ccs文件都是现成的，方便快捷，有利于装逼 身心健康

2333333~

Hexo 使用 Markdown（或其他渲染引擎）解析文章，使用 Node.js 渲染页面，通常在很短的时间内即可利用靓丽的主题生成静态网页。同时 Hexo 也拥有强大的插件系统，支持 Jade, CoffeeScript 等诸多强大插件。

真的有利于我这种小白。

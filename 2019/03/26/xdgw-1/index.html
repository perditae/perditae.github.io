《汪丁丁：行为金融学的缘起——兼谈“幸福感”的来源》学习感想

发表于 2019-03-26 | 分类于 学习感想
字数统计: 991 字 | 阅读时长 ≈ 3 分钟

近日，在老师的推荐下，我再次将汪丁丁先生的《行为金融学的缘起——兼谈“幸福感”的来源》一文细细研读了一番。与前次相比，虽依旧未能完全了解汪丁丁先生的思想，但依旧饶有收获。

相比之前的管中窥豹，在对金融学有了一定的了解后，再看文章，我自身愈发的认识到了学海的宽广。对于汪丁丁先生的文章，以下是私的一些见解，如有异议，恳请斧正。

汪丁丁先生在文章中有关经济学基本问题的论述中，指出了“价格决定什么”比“什么决定价格”更重要的观点，这也是汪丁丁先生的一个主要的观点，也是近代价格理论大师艾智仁的观点。我觉的这个观点太好了，历久以来，经济学对市价的分析，往往都是着重于价格是怎么决定的。在自由市场上，价高者得，而促成这个市价的游戏规则，则是私有产权的制度，市价就是确定成败的准则。在这样的大环境下，价格决定什么，是比什么决定价格更有价值的。

主流价格理论主要讲述价格如何决定，然而价格决定什么远比价格是如何决定的重要。

价格反过来也会决定如何生产、如何分配、如何消费等诸多方面。一些学者只会把吃饭问题看作是粮食、土地问题，难免会担忧粮食不够吃。而正确运用了价格理论，你就知道吃饭问题其实不是粮食、土地问题，而是收入问题。正确的经济学的看法是，一切的数量问题，本质上都是一个价格问题。

离开价格，我们不知数量为何物。你问我有多少粮食，那我要问你，出得起多少价啊？假如你愿意出一百块钱买一斤大米，那么我敢保证，家家户户阳台上、睡房里种的都是水稻，粮食会多得吃不完。实际上，你都不用出到一百块钱，只需出到汽油的价格，就能从汽车口中抢回吃不完的粮食。

价格此时的作用就是传递了有关市场的信息，以此来调整整个市场的均衡。挨饿不是因为粮食少，而是因为贫穷出不起价。发达国家农业不是主产业，可是发达国家的人民偏偏没有挨饿，发展中国家多数以农业为主产业，可是饥饿偏偏发生在这些以农业为主产业的国家。

在自由市场下，价格通过引导稀缺资源流向其最能发挥其价值的地方，从而起到调节供求，实现市场均衡。而在价格受到管制的情况下，虽然价格也能调节供求，实现供求均衡。但由于价格受到扭曲，失去了传递经济信号、引导稀缺资源流向的作用。从而导致租值消散，减少了社会福利。这就很好的体现了价格决定问题的价值。

价格可以指导我们生产、交易、分配、消费、避免租值消散、确定效率高低等等等等，价格就像生物的DNA传递的是一种信息，人们应该对价格保持敬畏之心，研究价格是如何传递信息的，以及传递什么样的信息——即价格决定了什么；如果不管价格决定带来的价值，而一味要去研究价格是如何被决定的，试图去改变价格，最终人们看似改变了货币价格，却遭到了“非货币价格”所带来的惩罚。
